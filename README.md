# Pi Homelab: DNS + VPN

## Hardware
- Raspberry Pi 4B 2GB
- Passive heatsink case
- Netgear RAX120v2 router
- Ethernet connection

## Services
- **Pi-hole** — Network-wide DNS filtering
- **WireGuard** — Personal VPN for remote access
- **Grafana + InfluxDB** — System monitoring dashboard
- **Nmap** — Network device inventory (manual scans)

## Performance Specs
- Idle: 25-35% CPU, 700-900MB RAM
- Normal use: 40-55% CPU, 900MB-1.1GB RAM
- Peak load: 85-90% CPU, 1.4GB RAM
- Passive heatsink: Sufficient (no throttling)
